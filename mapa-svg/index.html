Os dados são do Portal de Dados Abertos do Tribunal Superior Eleitoral. O código em R para chegar ao JSON que alimenta esta plataforma está aqui. O cálculo do percentual dos votos válidos considera a quantidade de votos recebidas por todos os candidatos. Por isso, no primeiro turno, a soma dos dois candidatos exibidos não dá 100%. Este projeto optou por exibir no primeiro turno apenas os dois candidatos que disputaram o segundo turno das eleições.

Plataforma desenvolvida por Gabriela Caesar em junho de 2022 durante o Master no Insper, com ajuda de Tiago Maranhão.
